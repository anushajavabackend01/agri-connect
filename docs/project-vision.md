# AgriConnect

## Product Vision

AgriConnect is a full-stack microservices application designed to help farmers manage crop information and market prices.

## Objectives

- Learn Spring Boot Microservices
- Learn React JS
- Learn API Integration
- Learn GitHub Workflow
- Simulate Real Agile Development

## Target Users

- Farmers
- Agricultural Consultants
- Administrators

## Business Features

- Farmer Management
- Crop Management
- Market Price Tracking
- Dashboard & Analytics

## Technology Stack

Backend:
- Java 17
- Spring Boot
- MySQL
- Spring Cloud Gateway

Frontend:
- React JS
- Axios
- Bootstrap

## Architecture

React UI → API Gateway → Microservices → MySQL
